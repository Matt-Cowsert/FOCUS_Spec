# Data Generator

Human-readable name of the entity that generated the dataset instance. The Data Generator ensures the technical accuracy and delivery of the data.

DataGenerator property adheres to the following requirements:

* DataGenerator MUST be present in the [Data Generator](#datagenerator) object.
* DataGenerator MUST be of type String.
* DataGenerator MUST conform to [StringHandling](#stringhandling) requirements.
* DataGenerator MUST NOT be null.
* DataGenerator SHOULD reflect the entity that generated the [*FOCUS dataset*](#glossary:FOCUS-dataset).

DataGenerator

Data Generator
